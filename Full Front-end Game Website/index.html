About Panda

Panda is a simple but user interactive website built from nowhere "hard coded" by: Full Stack Web developer DAGBUI EMMANUEL HOPE, an Aerospace Engineer. Panda was created on September 10th, 2023. This website is just for having fun as it comprise of three (3) different games. One, which is the most upgraded "NUMBER GUESSING GAME" and the other, "FIX THE SCRAMBLED WORD" and the last is a new game invented by Boss Panda, "PANDA FAST CLICKS". The guess the number game is about a number generated RANDOMLY between "0" and "50" but is hidden from you. So you are to guess the number before the timer stops or you lose! To win by distiction and to get the attention of Boss Panda, you are to get a perfect score "100" or "95". If you get anything below these scores, its just a normal win for you. The same is said for the fix the word game but instead of numbers, words are randomly generated and scrambled. The panda fast clicks game is about a box with a particular background-color, which changes depending on the mode of the game. You are to click that particular box before the color leaves that box and you should do this before the timer stops or you lose! Have fun! in the bamboos.
